The Game of Life

A Webassembly game written with Rust and Javascript

Rules:

  • Any living cell with fewer than two living neighbours dies
  • Any living cell with two or three living neighbours lives onto the next generation
  • Any living cell with more than three live dies
  • Any dead cell with exactly three living neighbours

Instructions:

  1. Create a grid
  2. Use mouse to draw the initial living cells
  3. Press play/pause to see the generations progress
  4. Change the update interval to speed up or slow down

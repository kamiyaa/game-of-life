The Game of Life

A Webassembly game written with Rust and Javascript

Rules:

  • Any live cell with fewer than two live neighbours
  • Any live cell with two or three live neighbours
  • Any live cell with more than three live
  • Any dead cell with exactly three live neighbours

Instructions:

  1. Create a grid
  2. Use mouse to draw all initial live cells
  3. Press play/pause to see the generations progress
  4. Change the update interval to speed up or slow down
